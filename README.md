# Description

Простое информационнное приложение. Пользователь выбирает
страну, интересующий его университет или колледж и переходит по ссылке 
за дополнительной информацией.

# Techologies

Язык Kotlin
Работа с базовыми элементами:
- XML;
- Image;
- Animations;
- Styles;
- Intents;

- Arrays;
- Collections;
- Interfaceces;

- JSON objects;
- Free API;
- RX Java;
- Retrofit;

Сохранение данных:
- Room;
